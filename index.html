Meteor

An Open-source Isomorphic JavaScript Web Framework.
It is a cohesive development platform with a collection of libraries
and packages that are bound together to build extraordinarily
dynamic web pages and mobile applications.

DDP

A Client-Server Protocol based on WebSockets for queuing and updating a Server-Side database and for synchronizing such updates among clients.

Built-In MongoDB

Meteor's built-in support of MongoDB makes it easier for applications that need a flexible architecture, cloud server support and real-time updates.

Other Databases

Meteor can be intergrated with distributed databases like RethinkDB, graph databases like Neo4j, relational databases like PostgreSQL, AlaSQL, Oracle.
