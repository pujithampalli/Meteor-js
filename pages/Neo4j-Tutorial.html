Meteor - Neo4j Tutorial

Neo4j can be easily integrated within few steps with Meteor. When an application is developed in Meteor with Neo4j as backend, a reactive and similar relationship layer is created, which helps the application to run smoothly.

In this tutorial we are going to learn how Meteor applications can be built and deployed by accessing its data stored in Neo4j database. Following are the process steps of how it can be done in command prompt.

Installation Steps

Step-1: Install Neo4j database in your local system.(First install 'brew' if you dont have!)

  • brew install neo4j

    npm install -g neo4j

Step-2: Configure Neo4j in the system and set the password.

  • $ neo4j start

Open this link in the browser.

  • http://localhost:7474/

Login with default username and password 'neo4j' and then set the password. Neo4j starts running in your system.

Working with Application

Step-3: Create Meteor application with the following command.

  • $ meteor create AppName

Step-4: You can add different packages based on your requirement. Following are example packages.

  • $ meteor add ccorcos:neo4j

    $ meteor add ostrio:neo4jreactivity

    $ meteor add ostrio:neo4jreactivity

Step-5: Now configure the application so that it uses Neo4j by creating 'lib' directory with 'neo4j.js' file in it inside your application. Add the following code in your 'neo4j.js' file to complete the configuration.

  • Meteor.neo4j.allowClientQuery = true; #to allow client query execution

    Meteor.neo4j.connectionURL = 'http://neo4j:password@localhost:7474'; #URL to Neo4j

    Meteor.neo4j.set.deny(neo4j.rules.write); #to deny all writing actions on client

Step-6: Now your Meteor application is running with Neo4j database. When you are done using the application make sure that you stop Neo4j database, else it will run in the background continuosly. You can stop it by using the following command.

  • neo4j stop
