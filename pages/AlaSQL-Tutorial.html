Meteor - AlaSQL Tutorial

AlaSQL is an open source database for relational and schemaless data which has strong SQL like environment. Its main focus is on query speen and data source flexibility. It provides with the same SQL statements which we commonly use in the backend, like SELECT, INSERT, UPDATE, CREATE, JOINS etc. It has many other features like,

  • Fast data processing for applications running on clients.
  • Data can be imported, exported or manipulated in several formats.
  • It supports all browsers and mobile applications.

    • In this tutorial we are going to learn how Meteor applications can use AlaSQL database. Following are the process steps of how it can be done.

    Installation Steps

    AlaSQL can be installed in following ways.

    1. Install AlaSQL with npm.

    • npm install --save alasql

    2. Install AlaSQL with bower.

    • bower install --save alasql

    3. Import AlaSQL from npm by adding the following line in your code.

    • import alasql from 'alasql';

    4. Install AlaSQL through command line with npm.

    • npm install -g alasql

    Example Queries

    You can access AlaSQL database in your code by adding the following line your in HTML file.

    • <script src="alasql.js"></script>

    Following are the sample quesries to create table, insert into table, select from table.

    1. Create table.

    • alasql("CREATE TABLE meteor (project_name string, dependencies number)");

    2. Instert values into table.

    • alasql("INSERT INTO meteor VALUES ('todos',5),('RocketChat',3),('Dr.Mongo',7);

    3. Select from table.

    • console.log( alasql("SELECT * FROM meteor WHERE dependencies > 1") );
