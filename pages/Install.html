Installing Meteor

Meteor is supported on all major operating systems, including:
• Mac: OS X 10.7 and above
• Windows:
      o Windows 7
      o Windows 8.1
      o Windows Server 2008
      o Windows Server 2012
• Linux: x86 and x86_64 systems


Windows users :
visit chocolatey.org/install and Install it. Chocolatey is required to Install Meteor.
Once Chocolatey is Installed run the below command using an Administrator command line

  • choco install meteor


    • Mac OS X or Linux users :
    open command line copy paste the below command and tap enter/return key

  • curl https://install.meteor.com/ | sh

    • This command will:
       • Connect to “install.meteor.com”.
       • Downloads the latest version of Meteor.
       • Installs that version of Meteor.
    This could take some time. Please be patient. And, if at all it asks you for the password, enter your password this is for authentication i.e., confirming that you are the system admin




































    Meteor Installation completed.
